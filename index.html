☁️ CONFIGURAÇÃO GOOGLE DRIVE

Espaço Ilimitado para Fotos

🎯 POR QUE USAR O GOOGLE DRIVE?

  • ✅ 15 GB GRÁTIS (espaço para ~15.000 fotos)
  • ✅ Armazenamento ilimitado com conta paga
  • ✅ Fotos em alta qualidade sem compressão
  • ✅ Acesso de qualquer dispositivo
  • ✅ Backup automático e seguro
  • ✅ Compartilhamento fácil de relatórios

📝 COMO CONFIGURAR (4 PASSOS SIMPLES)

📌 PASSO 1: Criar Projeto no Google Cloud
  1. Acesse: console.cloud.google.com
  2. Clique em "Criar Projeto"
  3. Nome: Shoemix Fotos
  4. Clique em "Criar"
📌 PASSO 2: Ativar API do Drive
  1. No menu, vá em "APIs e Serviços" > "Biblioteca"
  2. Pesquise: Google Drive API
  3. Clique na API e em "Ativar"
📌 PASSO 3: Criar Credenciais OAuth
  1. Vá em "Credenciais" > "+ Criar Credenciais"
  2. Escolha "ID do cliente OAuth"
  3. Tipo: "Aplicativo da Web"
  4. Adicione em "Origens autorizadas": http://localhost e file://
  5. 📋 COPIE o Client ID gerado!
📌 PASSO 4: Criar API Key
  1. Ainda em "Credenciais" > "+ Criar Credenciais"
  2. Escolha "Chave de API"
  3. 📋 COPIE a API Key gerada!
  4. (Recomendado) Clique em "Restringir chave" e selecione apenas "Google Drive API"
📌 PASSO 5: Cole as Credenciais Aqui Abaixo

Agora é só colar o Client ID e a API Key nos campos abaixo e clicar em "Salvar e Conectar"!
